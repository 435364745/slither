## Preview

> Server side is unsupported yet, it is still a single game now!!! 

[http://whxaxes.github.io/slither/](http://whxaxes.github.io/slither/)

## Installation

```
npm install
```

## Develop

```
npm run dev
```

Open your chrome and visit [http://localhost:9999](http://localhost:9999)

## Release
```
npm run release
```
